## 7. Voice-agent script implications (synthesis)

- `[E]` **Tone**: warm, human, "just you" — never salesy/pushy. Match clinic's authentic voice; consistency of patient-facing voice is an explicit owner value ("cohesiveness in how we speak to patients").
- `[E]` **First-contact empathy**: scripts must assume the caller is "in a really vulnerable place during a really hard time." Lead with reassurance, not a sales funnel.
- `[E]` **Position the agent as the chore-doer / admin helper**, echoing the owner's own AI mental model — "do the chores so I can do the care."
- `[E]` **Reschedule/cancellation outreach** is an emotional burden owners avoid because it feels "thirsty/desperate." A neutral agent doing it tactfully removes that cost — a concrete wedge.
- `[E]` **Referral-handoff capture**: clinics live on referrals routed through *other clinics' front desks* via phone/fax/email. An agent that reliably catches and logs inbound referral calls protects the "faucet that never stops dripping."
- `[E]` **After-hours / staggered-hours coverage**: "it's not 9 to 5" — the agent's always-on value should be framed against irregular clinic schedules.
- `[E]` **Gap to fill from other bundles**: this bundle lacks hard verbatim no-show/missed-call/phone-volume complaints. Source those from bundles 1/3/4/5 before finalizing the wedge copy.
