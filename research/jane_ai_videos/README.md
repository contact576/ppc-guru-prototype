# Jane "AI at Jane" video transcripts — raw research input

**Why only this playlist.** The rest of Jane's 463-video channel is diminishing returns
(feature tutorials duplicate the help-guide we already mined; customer-story playlists
duplicate the 81 Radio Front Desk episodes + 144 stories already captured). The **"AI at
Jane" (7 videos)** playlist is the exception worth grabbing: the #1 strategic risk is Jane
moving into AI / our space (lead-capture, attribution), and these videos reveal how
AI-forward they are and where they're headed.

## Drop the export here

- **File:** `jane_ai_videos.json` (same free Apify YouTube-transcript method as Radio Front Desk)
- **Format:** JSON or JSONL, all fields (videoUrl, videoDetails.title, transcript)
- Upload via GitHub browser to branch `claude/wonderful-heisenberg-1xz6am`.

## How it's processed

A subagent mines it for competitive signal only: what AI features Jane ships/teases, whether
any touch lead-capture / marketing / attribution (would narrow our wedge), and their AI
positioning. Distilled into `JANE_TEARDOWN.md` (competitive-risk section). Raw stays here.
