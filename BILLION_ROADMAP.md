# Roadmap to $1B — staged plan, numbers, and the honest path

*The scaling plan for the practitioner growth platform: physio (Canada) beachhead → multi-vertical. Numbers are planning estimates (🟢 grounded / 🟡 modeled / 🟠 aspirational), built on the validated inputs (`MARKET_SIZE.md`, `PRACTITIONER_AD_SCRAPE.md` 44.5% run ads, `CUSTOMER_DISCOVERY.md`). Applies the founder style: each phase has a fast-start move + an architected-for-scale guardrail.*

## Pricing assumption (the engine of every number)
ACV band **$150–$400/mo → ~$1,800–$4,800/yr** (validated against GHL $97, Weave ~$399, NexHealth $299, Kickcall $59 voice-only, dental tools $300–500). Plan base case **~$3,000 ACV**. 80–95% gross margin (low-inference software; voice usage metered so it never bleeds).

## The path-to-$100M math (so the goal is concrete, not a slogan)
- **$10M ARR** ≈ 3,300 customers
- **$50M ARR** ≈ 16,700 customers
- **$100M ARR** ≈ 33,000 customers
- **$1B ARR** ≈ 330,000 customers
NA practitioner TAM (physio+chiro+dental+optometry+vet+med-spa) ≈ **~290k+ practices**; broaden to **all local-service SMBs** (GHL's market) = **millions**.
**Honest read:** $10–50M is reachable *within healthcare practitioners*. **$100M needs all-of-NA-healthcare + early local-service. $1B requires becoming the local-service growth platform (GHL-scale) or going global multi-vertical** — the far horizon, contingent on capital + team scale-up. The architecture decision (modular, vertical=config) is *what keeps that path buildable without rewrites.*

## The phases

### Phase 0 — Validate (Month 0–2) · $0 ARR · spend ~$5–20k 🟢
- **Goal:** prove demand + get LOIs before compliance spend.
- **Do:** AI-build the A→B→C demo on **synthetic data**; demo to your own clinics + warm network; finish 8–12 discovery interviews; **get ≥3–5 LOIs / "I'd pay $X/mo"**; the Jane integration brief.
- **Team:** you + dev + AI. **Gate to Phase 1:** real owners commit a real $/mo.

### Phase 1 — First paying physio clinics, Ontario (Month 2–6) · ~$15–50k ARR · spend ~$25–50k 🟢
- **Goal:** 5–15 paying physio clinics on the compliant stack.
- **Do:** move to BAA infra + Canadian residency; sign BAAs; lawyer review; launch ONE regime (Ontario/PHIPA); your own clinics = the proof-of-ROI showcase.
- **Scale guardrail:** build multi-tenant + vertical-as-config from line one (no physio hard-coding).

### Phase 2 — Scale physio across Canada (Month 6–18) · ~$0.3–1M ARR 🟡
- **Goal:** 100–300 clinics. CA physio = 68% run ads = dense beachhead.
- **Do:** content/YouTube/SEO engine (job-term SEO) + direct outreach + PPC-Guru-as-channel; the neighbourhood benchmark turns on as density grows.
- **Team:** maybe co-founder + first 1–2 hires. **This is where product-market fit is proven or not.**

### Phase 3 — US physio + add Dental (CA) (Year 2–3) · ~$2–7M ARR 🟡
- **Goal:** 700–2,300 accounts. Dental = a config clone (same modules) + pays more ($300–500/mo) + advertises hardest.
- **Do:** US compliance (HIPAA attestation when a deal needs it); **begin agency-enablement** (train agencies to deploy — GHL doesn't). First real expansion proof that "vertical = config" holds.

### Phase 4 — Multi-vertical practitioners, North America (Year 3–5) · ~$15–50M ARR 🟠
- **Goal:** chiro, optometry, vet, med-spa + US dental — each a config + a go-to-market under the umbrella.
- **This is the realistic strong outcome** (a $15–50M vertical-SaaS business — bigger than Pabau, in reach).

### Phase 5 — The $1B horizon (Year 5+) · $100M→$1B ARR 🟠 contingent
- **Path A:** broaden to **all local-service SMBs** (home services, salons, legal, etc.) — the GHL-scale market — using the same engine.
- **Path B:** dominate **all-of-healthcare** globally + move up-market (groups/DSOs).
- Requires capital, a real team, and a category-defining brand. Name it as the ambition; earn the right to it by nailing Phases 0–3.
