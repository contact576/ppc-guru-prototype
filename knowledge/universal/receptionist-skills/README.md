# Universal · Receptionist + call-handling skills  (EVERGREEN)

The backbone — same for every clinic, every region. How the agent runs a call: greet → triage
(discipline, new/returning) → reason (no clinical probing) → offer a concrete slot → capture
name+number always → confirm → escalate clinical/emotional to a human. Plus phone etiquette,
message-taking, the fallback ladder.

**Default tags:** `layer: universal`, `region: GLOBAL`, `freshness: evergreen` (no `review_by`),
`topic: receptionist-skill`.

(YouTube transcripts). Raw → `/research/frontdesk_training/`, distilled + tagged → here.
